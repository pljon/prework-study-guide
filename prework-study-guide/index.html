HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The src element contains where the visible content is being pulled from.
  • The meta element describes the data on the document or webpage.
  • The footer element contains information at the bottom of the webpage.
  • git status: checks what branch we are currently on
  • git pull: fetches changes from another repository or local branch
  • git push: stages the uploaded changes from a local repository to a remote repository to be committed
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds all modifications in the current branch for next commit
  • git commit -m: describes and commits the staged changes made in a local branch to the main branch
